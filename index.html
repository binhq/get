BinHQ download scripts

Instructions

The following instructions apply to most of the downloader scripts.

Download the software using the following command: 
$ curl -sL https://get.binhq.io/SCRIPT_NAME | [PARAMS] bash
Note: this site used to be served over HTTP. This has changed recently and HTTP is now redirected to HTTPS, hence the -L switch in the cURL command.

Parameters:
  • VERSION: release version (eg. 1.0.0/v1.0.0, default: latest)
  • DEST: destination folder (default: $PWD)
  • OS: operating system (auto-detected)
  • ARCH: architecture (auto-detected)
Project specific parameters:
  • [PROJECTNAME]_VERSION
  • [PROJECTNAME]_DEST
Project specific parameters override their general version.

GitBin

GitBin helps downloading binaries from Github's Releases page. 

$ curl -sL https://get.binhq.io/gitbin | bash

Travis CI config
One of GitBin's goals is to ease binary download in CI/CD pipelines when you have no control over the environment (eg. in case of Travis CI). 

before_install:
    - mkdir -p $HOME/bin
    - curl -sL https://get.binhq.io/gitbin | DEST=$HOME/bin bash
    - gitbin get owner/repo version -o $HOME/bin
